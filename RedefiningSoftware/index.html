Software for Time Well Spent

Joe Edelman // March 2015.

There's something wrong with how software treats us. Software shouldn't compete to maximize screentime, but to demonstrate lasting improvements in our lives. We need changes in many fields, but we're starting with designers. We're gathering members, giving talks, and running design events around the world. In a few months we'll publish a community gallery revealing a more optimistic future for personal software and OS-level screens. We'll follow the same procedure with other groups of practicioners: in metrics, social network design, OS foundations, and marketing. Join us.
